## Programme de test de la notion de CollectionViewSource

Ce programme permet de tester la notion de filtrage et de tri d'une collection.
L'objet CollectionViewSource permet d'ajouter ces deux notions, d'une part pour choisir les différentes options de tri
mais aussi de définir une méthode qui permet de décider si un objet présent dans la collection en mémoire
doit être visualisé graphiquement dans l'interface utilisateur.
